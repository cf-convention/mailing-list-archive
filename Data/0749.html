Dear Michael and Pepe

> * volume_attenuation_coefficient_of_radiation_in_sea_water:m-1
> it could also be expressed as:
> * volume_beam_attenuation_coefficient_of_radiation_in_sea_water:m-1
> which would differentiate it from:
> * volume_diffusive_attenuation_coefficient_of_radiation_in_sea_water:m-1
> In a nutshell, the difference is in the angles considered. Beam attenuation
> considers the attenuation of (semi-)coherent light in a forward direction,
> while diffusive attenuation considers the attenuation from a diffuse source,
> which allows for a lot more forward scattering to be included in the
> measurement.

I propose we should stick with the plain attenuation, since it appears to be in
common use (for atmosphere as well) and find some other way to draw this
distinction. (But if necessary we can change our mind later and use aliases.)
The explanation helps. How do you measure a diffuse attenuation?

I think we have agreed that
> * volume_backwards_scattering_coefficient_of_radiation_in_sea_water:m-1
is to be defined as an integral over backward angles, so it won't have a
coordinate specifying the angle.
