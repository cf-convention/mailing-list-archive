I would like to propose the Lambert Azimuthal Equal-Area map projection
for inclusion. This projection has the following parameters:
grid_mapping_name = lambert_azimuthal_equal_area
longitude_of_central_meridian
latitude_of_central_meridian
false_easting
false_westing

the proj4 code is laea
