Thanks Jonathan.

        I assumed that Karl had arranged for the corresponding convective and
total cloud names to be added when the original MIP table was created
last year. They are:

Convective:

standard_name: convective_cloud_area_fraction_in_atmosphere_layer
PCMDI name: clc
units: %
long_name: Convective Cloud Fraction

standard_name: mass_fraction_of_convective_cloud_liquid_water_in_air
PCMDI name: clwc
units: 1
long_name: Convective Cloud Liquid Water (Mass Fraction)

standard_name: mass_fraction_of_convective_cloud_ice_in_air
PCMDI name: clic
units: 1
long_name: Convective Cloud Ice Water (Mass Fraction)

Total cloud (i.e. convective+stratiform):

standard_name: cloud_area_fraction_in_atmosphere_layer
PCMDI name: cl
units: %
long_name: Cloud Fraction

standard_name: mass_fraction_of_cloud_liquid_water_in_air
PCMDI name: clw
units: 1
long_name: Cloud Liquid Water (Mass Fraction)

standard_name: mass_fraction_of_cloud_ice_in_air
PCMDI name: cli
units: 1
long_name: Cloud Ice Water (Mass Fraction)

Cheers,
Keith
