Dear All,

If you have an interest in the area of CF standard names and haven't
hit 'list saturation' then I would like to invite you to subscribe to
the EnParDis (enabling parameter discovery) list on
http://mailman.nerc-bidston.ac.uk/mailman/listinfo/enpardis

EnParDis is a 12-month project to develop the BODC oceanographic
parameter dictionary in two ways:

To increase the population of terms from the present level of 9000 up
to 15000 based on mappings against other similar parameter dictionaries
in the oceanographic domain.

To develop new ways of implementing code classifications from the
current simple hierarchies towards an ontology

It is also planned to use some of the resources available to 'clean up'
the existing dictionary entries by standardising naming syntax and
improving the clarity of descriptions.

Although this dictionary is in the oceanographic domain there is
inevitably overlap with the atmospheric domain. Already there are over
100 atmospheric chemistry and meteorological codes in place.

The list will initially be used to circulate information on project
progress and advertise dictionary resources that we have made available
but once a user community is established I hope that the list will
become a mechanism for feedback on dictionary usage and development.

Cheers, Roy Lowry
