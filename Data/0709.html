Dear all

No further comments and suggestions were made, so the following have been
added to the standard name table. They are mostly to accommodate the needs of
the Paleoclimate Modelling Intercomparison Project Phase II and the EU MOTIF
project:

downward_sea_ice_basal_salt_flux:kg m-2 s-1
surface_eastward_sea_water_velocity:m s-1
surface_northward_sea_water_velocity:m s-1
water_evaporation_flux_where_sea_ice:kg m-2 s-1
heterotrophic_respiration_carbon_flux:kg m-2 s-1
litter_carbon_content:kg m-2
surface_downward_heat_flux_in_sea_water:W m-2
surface_net_downward_longwave_flux_assuming_clear_sky:W m-2
surface_net_downward_shortwave_flux_assuming_clear_sky:W m-2
toa_net_downward_shortwave_flux_assuming_clear_sky:W m-2
water_evaporation_flux_from_canopy:kg m-2 s-1
water_evaporation_flux_from_soil:kg m-2 s-1
water_sublimation_flux:kg m-2 s-1
soil_respiration_carbon_flux:kg m-2 s-1
product_of_eastward_sea_water_velocity_and_salinity:m s-1
product_of_eastward_sea_water_velocity_and_temperature:K m s-1
product_of_northward_sea_water_velocity_and_salinity:m s-1
product_of_northward_sea_water_velocity_and_temperature:K m s-1
biomass_burning_carbon_flux:kg m-2 s-1
integral_of_sea_water_temperature_wrt_depth_in_ocean_layer:K m
ocean_integral_of_sea_water_temperature_wrt_depth:K m
water_volume_transport_into_ocean_from_rivers:m3 s-1
northward_atmosphere_water_transport_across_unit_distance:kg s-1 m-1
eastward_atmosphere_water_transport_across_unit_distance:kg s-1 m-1
northward_atmosphere_dry_static_energy_transport_across_unit_distance:W m-1
eastward_atmosphere_dry_static_energy_transport_across_unit_distance:W m-1
sea_ice_transport_across_line:kg s-1
surface_upward_sensible_heat_flux_where_sea:W m-2
water_potential_evaporation_flux:kg m-2 s-1

These proposed standard names result from consultation with a number of
people, including Michel Crucifix, Jean-Yves Peterschmitt, Ian Ross, Olivier
Marti and Thierry Fichefet. There are a number of changes from the names
which currently appear in the PMIP2 CMOR tables, as follows:

biomass_burnt ->
  biomass_burning_carbon_flux (kg m-2 s-1)
depth_of_20deg_isotherm ->
  depth (m) with a single-valued coordinate of sea_water_temperature=20 degC
heat_content_of_ocean_layer ->
  integral_of_sea_water_temperature_wrt_depth_in_ocean_layer (K m)
  with a single-valued coordinate of depth whose bounds specify the layer
heat_content_to_300m ->
  integral_of_sea_water_temperature_wrt_depth_in_ocean_layer (K m)
  with a single-valued coordinate of depth whose bounds are 0 and 300 m
integral_of_product_of_eastward_wind_and_dry_static_energy_wrt_height ->
  eastward_atmosphere_dry_static_energy_transport_across_unit_distance (W m-1)
integral_of_product_of_eastward_wind_and_water_wrt_height ->
  eastward_atmosphere_water_transport_across_unit_distance (kg s-1 m-1)
integral_of_product_of_northward_wind_and_dry_static_energy_wrt_height ->
  northward_atmosphere_dry_static_energy_transport_across_unit_distance
  (W m-1)
integral_of_product_of_northward_wind_and_water_wrt_height ->
  northward_atmosphere_water_transport_across_unit_distance (kg s-1 m-1)
litter_carbon ->
  litter_carbon_content (kg m-2)
net_primary_productivitiy_of_carbon ->
  net_primary_productivity_of_carbon (kg m-2 s-1) (it was mispelled)
potential_evaporation ->
  water_potential_evaporation_flux (kg m-2 s-1)
product_of_eastward_current_and_salinity ->
  product_of_eastward_sea_water_velocity_and_salinity (m s-1)
product_of_eastward_current_and_temperature ->
  product_of_eastward_sea_water_velocity_and_temperature (K m s-1)
product_of_northward_current_and_salinity ->
  product_of_northward_sea_water_velocity_and_salinity (m s-1)
product_of_northward_current_and_temperature ->
  product_of_northward_sea_water_velocity_and_temperature (K m s-1)
river_discharge ->
  water_volume_transport_into_ocean_from_rivers (m3 s-1)
sea_ice_production ->
  tendency_of_sea_ice_thickness_due_to_thermodynamics (m s-1)
sea_ice_transport_at_fram_strait ->
  sea_ice_transport_across_line (kg s-1)
sea_surface_eastward_sea_water_velocity ->
  surface_eastward_sea_water_velocity (m s-1)
sea_surface_northward_sea_water_velocity ->
  surface_northward_sea_water_velocity (m s-1)

These are not needed because "where_land" is unnecessary and the names
without this clarification are already in the CF table:
  surface_snow_area_fraction_where_land
  surface_snow_melt_flux_where_land

It was agreed that these are not required for the moment:
  sea_ice_lead_area
  sea_ice_snow_volume

Clarification is still awaited about what the following mean:
  fraction_photosynthetic_active_radiation
  water_flux_into_ocean_for_free_surface
  product_of_eastward_eiv_current_and_salinity
  product_of_eastward_eiv_current_and_temperature
  product_of_northward_eiv_current_and_salinity
  product_of_northward_eiv_current_and_temperature

The names referring to "pft"="plant functional type" require a subsidiary
list of such types to be defined.

There should be no need for a standard_name of "hemisphere" since hemispheres
can be defined by coordinate bounds alone.

Cheers

Jonathan
