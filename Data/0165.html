Dear All

A program to check the compliance of netCDF files with the CF standard has been
written by Rosalyn Hatcher at the Hadley Centre, with support from the EU PRISM
project. Ag Stephens has installed it at the British Atmospheric Data Centre
and it can be accessed through a web form he has set up. This web page is
advertised on the CF home page http://www.cgd.ucar.edu/cms/eaton/cf-metadata/.
The program checks a netCDF file which you supply against the CF conformance
requirements and recommendations, as stated in another document linked from the
CF home page.

The CF-checker is a beta-release and development work is currently underway. If
you have suggestions for improvement then please e-mail Rosalyn Hatcher
(rosalyn.hatcher at metoffice.com).

Cheers

Jonathan
